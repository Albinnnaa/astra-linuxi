Astra Linux

Российская операционная система для защищённых сред

Что такое Astra Linux?

Astra Linux — это операционная система специального назначения на базе ядра Linux, созданная для нужд:

  • Органов государственного управления
  • Военного управления
  • Учреждений, работающих с информацией ограниченного доступа

История разработки

2008
Начало разработки АО «НПО РусБИТех» с участием Минобороны РФ
2009
Первая версия Astra Linux Common Edition ("Орёл")
2011
Выпуск Astra Linux Special Edition 1.2 ("Смоленск")
2013
Принята на снабжение Минобороны России
2024
Текущая версия Astra Linux Special Edition 1.8

Типы дистрибутивов

Common Edition

  • Свободно распространяемый
  • Несертифицированный
  • Для общего использования
  • Код: 2 (x86-64)

Special Edition

  • Сертифицированная ОС
  • Встроенная защита
  • Для защищённых систем
  • Обработка секретной информации
